# webautomation
this repo is used as repository of test web automation

Tools Used:
-----------

source 'http://rubygems.org'

1. ruby 2.6
2. watir, '~> 6.16', '>= 6.16.5'
3. rspec, '~> 3.9'
4. cucumber, '~> 3.1', '>= 3.1.2'

### Windows Instructions

1. Go to the path explorer '..\mekarisdet\features' and run this command line 'bundle install'
2. Go to path explorer '..\mekarisdet\features\step_definitions\Login' and run the test case using 'ruby ${nameFile}. ex: 'ruby Valid_Login.rb'
